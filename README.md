# ПредставлениеAi — МВД РК

Платформа юридического помощника на базе ИИ для Министерства внутренних дел Республики Казахстан. Загрузка материалов уголовных дел, общение с ИИ-ассистентом, генерация процессуальных документов.

## Возможности

- **Уголовные дела** — Регистрация дел по номеру ЕРДР с указанием фабулы
- **Загрузка документов** — PDF, DOCX, TXT, изображения (до 50 МБ) с прогрессом загрузки
- **RAG-чат** — Вопросы по материалам дела с контекстным поиском (Saiga NeMo 12B)
- **Пользовательские шаблоны** — Загрузка DOCX/RTF/ODT шаблонов через админ-панель
- **Генерация документов** — По загруженным шаблонам с фактами из дела
- **WYSIWYG-редактор** — TipTap с шрифтами, размерами, ссылками, цитатами, выделением
- **ГОСТ-экспорт DOCX** — Times New Roman 14pt, 1.5 интервал, поля по ГОСТ
- **Saiga NeMo 12B** — Русскоязычная модель на базе Mistral NeMo (IlyaGusev/saiga_nemo_12b)
- **Логотип МВД РК** — В верхнем баре навигации
- **Роли** — Администратор (первый пользователь) и Следователь

## Технологии

| Слой | Технология |
|------|-----------|
| Фронтенд | Next.js 15, React 19, TypeScript, Tailwind CSS |
| Редактор | TipTap (шрифты, ссылки, цитаты, выделение, выравнивание) |
| Бэкенд | FastAPI, Python 3.12, SQLAlchemy async |
| БД | PostgreSQL 16 |
| Вектор | ChromaDB |
| ИИ | Saiga NeMo 12B (OpenAI-совместимый API) |

## Быстрый старт

```bash
# Запуск Saiga NeMo 12B через Ollama
ollama run saiga_nemo_12b

# Запуск платформы
docker compose up --build
```

| Сервис | URL |
|--------|-----|
| Фронтенд | http://localhost:3000 |
| Бэкенд | http://localhost:8000 |
| PostgreSQL | localhost:5432 |

## Работа с системой

1. **Регистрация** → первый пользователь = администратор
2. **Загрузка шаблонов** (админ) → `/templates` → загрузить DOCX/RTF шаблоны
3. **Создание дела** → указать номер ЕРДР и фабулу
4. **Загрузка файлов** → PDF, DOCX, TXT через боковую панель
5. **Чат с ИИ** → вопросы по материалам дела
6. **Генерация** → выбрать шаблон → ИИ заполняет по фактам
7. **Редактирование** → WYSIWYG редактор → экспорт DOCX

## Переменные окружения

| Переменная | Описание | По умолчанию |
|-----------|---------|-------------|
| `AI_MODEL` | Название модели | `saiga_nemo_12b` |
| `AI_API_URL` | OpenAI-совместимый эндпоинт | `http://host.docker.internal:11434/v1/chat/completions` |
| `MAX_UPLOAD_BYTES` | Максимальный размер файла | `52428800` (50 МБ) |
| `CORS_ORIGINS` | Разрешённые источники | `http://localhost:3000,http://127.0.0.1:3000` |
